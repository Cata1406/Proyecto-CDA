## Corra localmente

Clone el proyecto

```bash
  git clone https://github.com/Cata1406/Proyecto-CDA
```

Vaya al directorio del proyecto

```bash
  cd Proyecto-CDA
```

Instale las dependencias

```bash
  pip install -r requirements.txt
```
